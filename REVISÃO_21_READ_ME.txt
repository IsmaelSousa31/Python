Encontrar o Maior Número em uma Matriz
Este é um programa simples em Python que encontra e imprime o maior número em uma matriz.

Funcionalidade
O programa itera sobre os elementos de uma matriz bidimensional e identifica o maior valor presente nela.

Como Usar
Execute o programa em um ambiente Python.
O programa imprimirá o maior número encontrado na matriz.
Observações
A matriz é definida diretamente no código. Se desejar, você pode modificar a matriz para testar com diferentes conjuntos de números.
O programa é projetado para funcionar com matrizes de tamanho fixo (3x3). Se desejar, você pode modificar o código para trabalhar com matrizes de diferentes tamanhos.
