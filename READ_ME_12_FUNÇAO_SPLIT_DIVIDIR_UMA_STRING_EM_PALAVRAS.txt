Objetivo: Separar uma string em substrings com base em um delimitador (espaço em branco) e armazená-las em uma lista.
Este programa em Python demonstra como usar a função split() para dividir uma string em palavras, utilizando o espaço em branco como delimitador, e armazenar essas palavras em uma lista.

Como funciona o código:
A variável frase contém a string que será dividida em palavras.
A função split() é aplicada à string frase, sem especificar nenhum argumento. Por padrão, essa função divide a string em substrings com base nos espaços em branco (espaços, tabulações, quebras de linha, etc.).
O resultado da operação é armazenado na lista lista_frases.
A lista resultante é então impressa na tela, mostrando as palavras separadas.
