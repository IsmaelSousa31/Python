Gerenciador de Lista
Este é um programa simples em Python que permite ao usuário inserir, listar e apagar itens de uma lista dinâmica.

Funcionalidades
O programa oferece as seguintes opções:

Inserir (i): Permite ao usuário inserir um novo item na lista.
Listar (l): Exibe todos os itens atualmente presentes na lista, juntamente com seus índices.
Apagar (a): Permite ao usuário selecionar e remover um item da lista com base no seu índice.
Como Usar
Execute o programa em um ambiente Python.
Selecione uma das opções fornecidas:
Para inserir um item, pressione 'i', digite o nome do item e pressione Enter.
Para listar todos os itens, pressione 'l'.
Para apagar um item, pressione 'a', digite o índice do item a ser removido e pressione Enter.
Siga as instruções fornecidas pelo programa para cada opção selecionada.
Observações
O programa não permite inserir opções inválidas. Se uma opção inválida for inserida, o programa solicitará que o usuário insira novamente.
Quando o usuário tenta apagar um item da lista, o programa verifica se o índice fornecido é válido.
Se ocorrer uma exceção durante a execução do programa, uma mensagem de erro será exibida.
