Implementação
Entrada de Dados
O programa solicita ao usuário que insira dois valores.
Comparação de Valores
Os valores inseridos são convertidos para inteiros.
O programa verifica se o primeiro valor é maior que o segundo.
Se sim, exibe uma mensagem indicando que o primeiro valor é maior.
Se não, verifica se o segundo valor é maior que o primeiro.
Se sim, exibe uma mensagem indicando que o segundo valor é maior.
Se nenhum dos casos acima for verdadeiro, os valores são iguais e uma mensagem indicando isso é exibida.
