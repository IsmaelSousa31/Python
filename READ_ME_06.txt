Adicionando e Iterando uma Lista em Python
Este código demonstra como adicionar um elemento a uma lista e como iterar sobre os elementos da lista.

Funcionamento do Programa
Inicialmente, uma lista lista é criada com os nomes 'Maria', 'Helena' e 'Luiz'.
Um novo nome, 'João', é adicionado à lista usando append.
O código itera sobre a lista usando range(len(lista)) para obter os índices e for para imprimir cada elemento e seu tipo.
