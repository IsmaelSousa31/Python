Calculadora de IMC em Python
Este é um simples programa em Python que calcula o Índice de Massa Corporal (IMC) com base na altura e peso fornecidos.

Exemplo de Uso
python
Copy code
nome = 'João Batista'
altura = 1.76
peso = 66
imc = peso / (altura * altura)

print(nome, 'tem', altura, 'de altura,')
print('pesa', peso, 'quilos e seu IMC é' )
print(imc)
Saída
Copy code
João Batista tem 1.76 de altura,
pesa 66 quilos e seu IMC é 21.282949105995163
