## Verificação de Número Inteiro e Paridade
Este trecho do código solicita ao usuário que insira um número inteiro. Em seguida, verifica se o número inserido é um número inteiro. Se o número for inteiro, o programa também verifica se é par ou ímpar.

- Número Inteiro: O programa verifica se o número inserido é um número inteiro.
  
- Paridade: Se o número for par, o programa imprime que o número é par. Caso contrário, imprime que o número é ímpar.

Caso o valor inserido não seja um número inteiro, o programa captura a exceção `ValueError` e imprime a mensagem 'O numero não é do tipo inteiro'.
