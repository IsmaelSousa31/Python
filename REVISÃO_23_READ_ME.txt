Calculadora de Média de uma Matriz

Este programa em Python calcula a média dos elementos de uma matriz. No exemplo fornecido, a matriz m é uma matriz 3x3, preenchida com valores constantes. O código itera sobre cada elemento da matriz, soma-os e, em seguida, divide essa soma pelo número total de elementos na matriz para obter a média.

Como funciona:

Uma matriz m é definida, com valores constantes.
Dois laços for são usados para iterar sobre cada elemento da matriz.
A variável media é usada para acumular a soma de todos os elementos.
A variável n conta o número total de elementos na matriz.
Após o término da iteração, a média é calculada dividindo-se a soma acumulada pelo número total de elementos.
A média resultante é impressa na saída padrão.
