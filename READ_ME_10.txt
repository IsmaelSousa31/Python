Verificação de Palíndromos em Python
Este é um programa simples em Python que verifica se uma palavra inserida pelo usuário é um palíndromo. Um palíndromo é uma palavra que é lida da mesma forma de trás para frente.

Funcionamento do Programa
Entrada da Palavra: O programa solicita ao usuário que digite uma palavra.
Inversão da Palavra: A palavra digitada é invertida para verificar se é um palíndromo.
Verificação do Palíndromo: O programa compara a palavra original com a palavra invertida para determinar se é um palíndromo.
Saída do Resultado: O programa exibe uma mensagem indicando se a palavra é ou não um palíndromo.
