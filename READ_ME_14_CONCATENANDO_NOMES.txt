Concatenar Nomes de uma Lista em uma Única String
Este programa em Python tem o objetivo de receber uma lista de nomes e concatená-los em uma única string, separando-os por vírgulas.

Pergunta:
Como você concatenaria os nomes de uma lista em uma única string, separando-os por vírgulas?

Como funciona o código:
A variável lista contém uma string com vários nomes separados por espaços.
A função split(' ') é aplicada à string lista, dividindo-a em substrings usando o espaço em branco como delimitador. O resultado é armazenado na lista string.
Em seguida, a função join() é utilizada para concatenar os elementos da lista string em uma única string, utilizando a vírgula como separador. O resultado é armazenado na variável stro.
Por fim, a string resultante, contendo os nomes concatenados e separados por vírgulas, é impressa na tela.
